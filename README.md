# MapMaker Lite

Editor leggero per mappe fantasy (worldbuilding) con PNG trasparenti.

## Avvio (web)
```bash
pnpm i
pnpm dev
```
Apri http://localhost:5173

## Build web
```bash
pnpm build
pnpm preview
```

## Desktop (Electron)
```bash
pnpm dev          # sviluppo con finestra desktop
pnpm build:desktop
```

## Funzioni
- Drag&drop asset (PNG/WebP/JPG)
- Griglia + Snap
- Layer/ordinamento, lock/visibilità
- Trasformazioni: posizione/scala/rotazione/opacità
- Esportazione PNG
